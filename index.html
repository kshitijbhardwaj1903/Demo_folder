Firstly we'll add our files, now we'll go on github and create new repositary. Then we'll look for a link in https. Copy that, then write our command git remote add origin (link).
As mentioned in our imp_point file, we can use git remove -v to verify ki humari files kaunsi repositary pe push horhi hai. We use git branch -M main to change our master branch name to main branch name. (Agar master likha aye to, wrna directly hi main likha aega.) Now we can push our code in github using git push origin main. We can make changes in our readme file or index file, and then use git add . then write git commit -m "name" then write git push origin main. Also, if we dont want to write git push origin main everytime, we can write 'git push -u origin main' once. And ab hum agar koi change kare, git add kare, then git commit kare, uske baad git push origin main likhne ki need nahi hai, we can only write git push.
